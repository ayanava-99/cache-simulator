# Write-Back Delayed Eviction Demo
# Designed to explicitly demonstrate Write-Back performance.

W 0x1000 0xAA
W 0x1000 0xBB
W 0x1000 0xCC

# The above 3 writes happen instantly in cache without touching the database!
# This read causes a conflict, forcing the dirty block to evict and write to the database.
